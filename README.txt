******* Описание *******
Данный бот может сохранять отправленные веб ссылки как в локальную (SQLite) так и в онлайн (Notion) базу данных.

* Если хотите сохранить ссылки также в Notion, требуется в начале отправить свои Notion API и Database ID, через кнопку "Add Notion link" в главном меню
* При добавлении Notion API, все последующие ссылки будут также добавлятся в Notion, предыдущие ссылки останутся в локальной базе данных (если только заново их не добавить)
    (В преспективе можно сделать, чтобы отправленние ссылки до добавления Notion API были добавлены в Notion)
* На данный момент можно использовать только один Notion API и Database ID

1. Отправка ссылок:
    Если отправлен текст с одной ссылкой, то эта ссылка автоматически сохранится в базе.
    В случае отправки текста с более чем одной ссылкой, выводится меню с этими ссылками. При выборе ссылки из списка, эта ссылка сохраняется в базе и меню обновляется с оставшиемся элементами (имеется кнопка "Add all links")
    В случае если веб страница не содержит поле <title>, то в поле Title сохраняется название ссылки (с приставкой http://).
    При сохранении ссылки - сохраняется его Telegram ID отправителя, URL, Title (если нет, то сам URL), Source (telegram username/channel) и время добавления в базу
        Такие параметры как Category и Priority возможно изменить в редактировании, автоматически не присваиваются

2. Получение списка ссылок:
    Список можно получить по следующим параметрам: Category, Priority, Source и все ссылки сразу
    При выборе ссылки, пользователь получает информацию об этой ссылки и меню с выбором изменения каких-либо параметоров

3. Редактирование ссылок
    * На данный момент редактирование касается только элементов в локальной базы данных
    * Если ссылка сохранена как в локальной так и в Notion, то изменения каснутся локальной (в преспективе - исправимо)
    Редактировать можно следующие параметры: URL, Title, Category, и Priority
        В случае изменения URL, автоматически меняется и Title, если Title отсутствует - то URL в качестве Title
    Возможно сохранить ссылку в Notion (если был добавлен Notion API) через кнопку "Add to Notion" (если ссылка была добавлена
	в локальную базу данных до добавления Notion API

4. Удаление ссылок
    * На данный момент удаление касается только элементов в локальной базы данных
    Удалить можно по одному элементу или сразу все ссылки пользователя ("Delete all links")
    

******* Запуск *******

Используйте токен вашего телеграм бота в config.py или через .env
Запустите __main__.py
